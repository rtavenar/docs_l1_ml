visu_donnees_synthetiques(X, clusters=None)#

Visualise un jeu de données synthétique (généré par charger_donnees).

Paramètres:

  • X (numpy.ndarray) – jeu de données (2D) à visualiser

  • clusters (list ou None) –

    • si None : ne pas colorer les points

    • sinon : clusters est une liste indiquant, pour chaque point du jeu de données, le numéro du cluster auquel il est rattaché
